# Analise-de-cancelamento

Esse sistema consiste na análise de um banco de dados de uma empresa que teve um alto número de cancelamento de serviços e na busca de possíveis respostas do por quê disso e busca por possíveis soluções para tal situação. Foram utilizadas as bibliotecas pandas para leitura, filtragem e análise do banco e plotlib para gerar gráficos e possibilitar a visualização desse estudo de dados , facilitando a procura de pontos que poderiam gerar os problemas de cancelamento.
